Fichiers avec Python et les notebook¶

  • A l’aide de Python, créer un fichier contenant le mot “Bonjour”

  • Ouvrir ce fichier à l’aide d’un éditeur de texte et modifier le.

  • Lire à nouveau le fichier avec Python

Solution¶

# Sous windows le bureau se trouve ici
PATH = r"C:\Users\login\Desktop" # On utilise le r"" pour que les \ ne soient pas interprétés
PATH = '/tmp' # J'utilise un repertoire temporaire sous linus

import os

# La commande os.path.join permet de créer un chemin indépendament du l'OS
nom_du_fichier = os.path.join(PATH, 'fichier.txt')

with open(nom_du_fichier, 'w') as f:
    f.write('Bonjour')
    
with open(nom_du_fichier) as f:
    print(f.read())
